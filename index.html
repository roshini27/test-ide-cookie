Google DoubleClick Cookie Sync Test Page

This page fires the official Google DoubleClick cookie sync pixel. If third-party cookies and ad consent are allowed, the IDE cookie will be created under the .doubleclick.net domain.

✅ How to Verify

  1. Open this page in Google Chrome.
  2. Make sure third-party cookies are NOT blocked.
  3. Open DevTools → Application → Cookies.
  4. Select https://cm.g.doubleclick.net
  5. Look for cookie name: IDE

❗ If You Don’t See the IDE Cookie

  • Third-party cookies are blocked
  • You are using Safari / Brave / Firefox
  • You are in Incognito Mode
  • Ad/Marketing consent is not given
